About

CohortEx extracts and analyzes cohorts for retrospective studies based on a parametrized SQL script. This is the very first version of a Jekyll site hosted on Github to edit study parameters and blog the study results. The parameter's editor enables the creation of new study parameter's set directly on GitHub.
